# 极简桌面记账本 (Flet 版)

## 项目简介
本项目是一个基于 Python Flet 框架开发的现代化桌面记账软件。相比传统桌面应用，它拥有 Web 级别的流畅交互体验和精美的 Material Design 界面，同时保持本地运行的数据安全性。

## 核心功能
1. **极速记账**：简洁的输入表单，支持金额校验与分类选择。
2. **交互式图表**：集成 Flet 原生图表组件，鼠标悬停即可查看各类支出详情。
3. **流水清单**：实时更新的收支列表，支持按时间倒序查看。
4. **本地存储**：数据仅保存在本地 JSON 文件，无云端上传，隐私绝对安全。
5. **筛选查看**：按“月份”和“分类”筛选记录。

## 技术亮点
- 表单验证：金额不能为负，日期必须合法
- 用 `filter` + `reduce`（等价算法）筛选/统计数据
- 第三方图表库本地化：Flet Chart组件

## 运行环境
- Python 3.8+
- flet（已安装）

## 快速开始
1. 安装依赖：
   ```bash
   pip install flet
   pip freeze > requirements.txt
   ```
2. 运行软件：
   ```bash
   python main.py
   ```

## 使用说明
- 在顶部输入金额、分类、备注和日期。
- 点击“记一笔”完成添加。
- 中间图表会实时更新显示各分类支出比例。
- 可以选择月份和分类进行筛选查看。
- 底部记录按日期倒序显示。

## 项目亮点
- 低代码高颜值：利用 Flet 声明式 UI，以极少的代码实现了现代化的交互界面。
- 数据可视化：动态饼图实时反映消费结构，辅助理财决策。
- 健壮性：内置完整的输入验证机制，防止非法数据录入。
