# 极简桌面记账本

## 项目简介
MiniLedger 是一款面向个人用户的桌面记账本应用，主打“轻量、直观、可追溯”。
支持收入/支出记录、分类管理、筛选统计、饼图可视化，并提供预算管理、固定模板、回收站与撤销、导入导出、自动备份与恢复等能力，适合日常财务管理与小型项目收支记录。

软件架构
软件架构说明：

1.表现层：基于 Flet 构建桌面图形界面，提供录入、查询、统计与管理页面。
2.业务层：实现收支记录、预算控制、模板自动入账、回收站、撤销、导入导出、备份恢复等逻辑。
3.数据层：采用本地 JSON 文件存储，保障离线可用与数据安全。
4.图表层：使用 Matplotlib 生成统计饼图并在界面中展示。

安装教程

1.安装 Python 3.10 及以上版本。
2.安装依赖：pip install flet matplotlib。
3.在项目目录运行：python main.py 启动应用。

使用说明

1.记账：在“支出”或“收入”输入区填写金额、分类、日期和备注，点击按钮完成记账。
2.管理：可设置预算、创建固定模板、使用筛选查询、查看图表统计，并通过回收站/撤销避免误操作。
3.数据：支持 JSON/CSV 导入导出，建议定期手动备份，必要时可从备份一键恢复。
