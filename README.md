# 极简记账本

## 项目简介
极简记账本是一款基于 Python 和 Flet 框架开发的本地桌面记账软件。应用主打轻量与极简，无繁杂的云端同步。所有的账单流水、模板与分类数据均以 JSON 格式安全保存在您的本地电脑中，为用户提供清晰、安全的个人财务管理体验。

## 主要功能
- **日常记账**：支持记录日常收入与支出，并允许根据个人习惯添加自定义收支分类。
- **图表统计**：内置基于 Matplotlib 的分类收支饼状图，直观展示各项花销的资金流向占比。
- **预算管理**：支持设定“月度总预算”以及具体到类目的“分类预算”，按月实时监测超支情况。
- **固定模板与定时**：支持设置固定的收支模板（例如每月固定的房租、工资），支持一键执行或记录每月定时处理。
- **数据安全与备份**：自带防误删“回收站”与撤销功能；在执行删除、导入等操作前会自动生成数据备份快照，并支持手动备份与快照恢复。
- **导入与导出**：支持将账本流水导出为 CSV 报表或 JSON 文件，同时支持从外部将标准的 CSV/JSON 文件导入账本。

## 运行环境
- **操作系统**：Windows系统 (Windows 10/11) / macOS / Linux 桌面环境
- **基础环境**：Python 3.8 及以上版本
- **核心依赖库**：
  - `flet` 
  - `matplotlib` 

## 安装教程
1. 请确保您的电脑已成功安装 Python 3.8+，并已配置好系统环境变量。
2. 下载或克隆本项目的源代码，并解压至本地文件夹中。
3. 打开系统的终端（Terminal）或命令提示符（CMD/PowerShell），进入该项目所在目录。
4. 在终端中运行以下命令，安装项目所需的全部依赖包：
```bash
pip install flet matplotlib

## 使用说明
- 在顶部输入金额、分类、备注和日期。
- 点击“记一笔”完成添加。
- 中间图表会实时更新显示各分类支出比例。
- 可以选择月份和分类进行筛选查看。
- 底部记录按日期倒序显示。

## 项目亮点
- 低代码高颜值：利用 Flet 声明式 UI，以极少的代码实现了现代化的交互界面。
- 数据可视化：动态饼图实时反映消费结构，辅助理财决策。
- 健壮性：内置完整的输入验证机制，防止非法数据录入。
